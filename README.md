# Refresh Road Trip

#### Requirements

- [Middleman](http://middlemanapp.com/)

#### Setting Up Middleman

- `gem install middleman`

#### Starting Middleman

- `cd refresh-road-trip`
- `middleman server`

The server is now running at [http://localhost:4567](http://localhost:4567/).

#### Compiling Middleman

- `middleman build`

#### Publishing

- `rake publish`
